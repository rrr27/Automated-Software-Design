MDELite 7 is a 5th generation codebase written in Java to illustrate MDE principles and tooling from a relational database perspective.  The core set of HTML documents are::
  • MDELite Installation and Overview Manual 
  • Constraint Programs -- how to write constraint checking programs
  • M2M Programs -- how to write Model-to-Model  transformations
  • M2T Programs -- how to write Model-to-Text programs
  • Error Listings -- a (reasonably) complete set of Errors issued by MDELite
The following documents are examples or more detailed MDELite concepts and applications.
  • Families -- the MDELite counterpart to "Families to Persons" example in ATL
  • Violet Manual -- an overview of Violet's connection to MDELite
  • Yuml Manual -- an overview of Yuml's connection to MDELite
  • Bootstrapping -- how MDELite is bootstrapped
  • Allegories -- the connection of relational databases to category theory and MDELite
